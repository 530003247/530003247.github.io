# localhost 127.0.0.1和本机ip三者的区别


```
localhost 
不联网 
不使用网卡，不受防火墙和网卡限制 
本机访问 

127.0.0.1 
不联网 
网卡传输，受防火墙和网卡限制 
本机访问 

本机IP 
联网 
网卡传输 ，受防火墙和网卡限制 
本机或外部访问
```
